DOCUMENTO DE REQUISITOS DE SOFTWARE PARA SISTEMA DE CONTACTOS

En el presente documento especifica como debería funcionar el software de gestión de contactos. 

REQUISITOS FUNCIONALES
La cabecera y el pie de pagina serán los mismos en todas las pantallas.

•	Inicio de Sesión
El software debe permitir a usuarios registrados iniciar sesión

•	Creación de cuenta
El software debe permitir a los usuarios crearse una cuenta sino tienen para poder acceder al aplicativo

•	Gestión de Usuarios
El software debe permitir al usuario agregar, modificar y eliminar contactos.
El usuario solo podrá ver los usuarios que el agregó
	
•	Bloqueo de Sesión
El sistema debe permitir al usuario bloquear su sesión, donde se solicitaría solo su contraseña para volver al menú de inicio.

Si estando la sesión iniciada, hay inactividad en el sitio por 15 minutos, la sesión se bloqueará automática.
